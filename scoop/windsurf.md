# How to install `Windsurf` with scoop

## Finding
```bash
PS C:\> scoop search windsurf
Results from local buckets...

Name     Version Source Binaries
----     ------- ------ --------
windsurf 1.10.5  extras
```

PS C:\> scoop install extras/windsurf
