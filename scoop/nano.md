# How to install `Nano` command

## Required
- 7zip

## Install `7zip`
```sh
scoop bucket add main
scoop install main/7zip
```

## Via `Scoop`

```sh
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
irm get.scoop.sh | iex
```
```sh
scoop bucket add .oki https://github.com/okibcn/Bucket
scoop install nano
```
